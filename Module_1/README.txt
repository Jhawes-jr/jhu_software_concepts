How to run

1) Create & activate a virtual environment (Windows PowerShell):
   python -m venv venv
   .\venv\Scripts\Activate.ps1

2) Install dependencies:
   pip install -r requirements.txt

3) Start the app:
   python run.py

Server runs on http://127.0.0.1:8080
